## Neural Networks
- Single Layer Perceptron (SLP)
- Multi Layer Perceptron (MLP)

## SLP architecture:
    - activation function: softmax
    - cost function: cross entropy

## MLP architecture:
    - 1st layer: sigmoid
    - 2nd layer: softmax
    - cost function: cross entropy

## Features:
    - Early stopping
    - Variable learning rate
    - Plot error vs iteration
